## Development

**Install deps**
```
npm install
```

**Install MongoDB**

Download here: https://www.mongodb.com/download-center

On a Mac
```
brew install mongodb
```

**Run the setup script**
```
npm run setup
```

**Generate Bundles** (this will also watch for changes):
```
npm run build
```

**Start the server**
```
npm start
```

Head to http://localhost:3200/

### Warning: Expect nightly breaking changes!
